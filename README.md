# Sentiment Analysis 

## Overview
This project performs sentiment classification on text data using a deep learning model (CNN–BiLSTM).

## Key Steps
- Data cleaning and preprocessing using Pandas and NumPy  
- Exploratory Data Analysis (EDA) and visualization  
- Model building using TensorFlow/Keras  

## File
- `sentiment_analysis_cnn_bilstm.ipynb`: contains full code, EDA, and model
